# Josefine — CV och Portfolio

En personlig CV- och portfoliowebbplats byggd med Next.js, TypeScript och Tailwind CSS. Projektet är avsett att presentera erfarenhet, projekt och kontaktuppgifter på ett modernt, responsivt sätt.

## Innehåll

- CV-sida: presentation, kontakt och kompetenser.
- Om mig: mer information om bakgrund och mål.
- Portfolio: projektvisning med kort och detaljerade sidor.

## Demo

Se en skärmbild i repositoryt: `screenshot.png`. Kör projektet lokalt för full interaktivitet.

## Teknisk översikt

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- React


## Komma igång (lokalt)

1. Klona repot:
   git clone https://github.com/knixan/josefinecvportfolio-repo.git
2. Gå till mappen och installera beroenden:
   npm install
   eller
   yarn install
3. Starta utvecklingsservern:
   npm run dev
   eller
   yarn dev
4. Öppna webbläsaren på: http://localhost:3000

## Struktur

- `app/` — Next.js App Router-sidor
- `components/` — återanvändbara React-komponenter (kort, header, footer)
- `public/` — statiska tillgångar (bilder, ikoner)
- `src/data/` — exempeldata för portfolio

## Distribution

Bygg för produktion:
npm run build
npm run start

Rekommenderad plattform för distribution: Vercel (Next.js optimerad).
